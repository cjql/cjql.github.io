# flaskr  
## templates  
### base.html文件  
`<title>{% block title %}{% endblock %}`：  
以此文档为基的其他文档，`{% block title %}{% endblock %}`里的内容都是title。  
- `{% block title %}{% endblock %}`：起占位作用。title可以是任意的html标签。如果static文件以外的文件里的html元素不用block包裹，则无法生效。  
- `<title></title>`：决定其内容是title。  

`<link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">`：  
- static是flaskr的一级子文件夹。  
- style.css是static里的文件。  

`<script type=text/javascript src="{{ url_for('static', filename='table.js') }}"></script>`：  
- 全局script。  

`<nav></nav>`：导航栏。  
`<a href="{{ url_for('blog.shixi')}}">图集</a>`：  
- blog是flaskr里的py文件  
- shixi是blog.py的一个路径装饰器下的函数。  

`{{ g.user['username'] }}`：用户名，网页刷新时自动传入。  
`{% if g.user %}{% else %}{% endif %}`：  
- 条件表达式。**是谁的呢**  
- {% if g.user %}：如果已登录。  

`<ul></ul>`：**通过css来控制对齐吗？**  
`<section class="content"></section>`：子html的格式定义。  
`{% for message in get_flashed_messages() %}`：**全项目只用到了flask.flash(error)。用于程序出错时，在页面显示错误。error可自定义。**  
`{{ message }}`：**flash错误信息**  
### blog  
#### blog.html  
`{% extends 'base.html' %}`：继承自base.html  
`<a class="action" href="{{ url_for('blog.create') }}">New</a>`：不传数据，只调用函数。  
`<a class="action" href="{{ url_for('blog.update', id=post['id']) }}">编辑</a>`：传数据，且调用函数。`id=post['id']) `用来传参。  
`{% for post in posts %}`：自动传post。post的内容在blog.py里面定义。  
`{% if g.user['id'] == post['author_id'] %}`：只显示当前登录用户的数据。  
`post['created'].`：获取post的键created的值  
`{}`：html内的逻辑语句或占位符。  
`{{}}`：抽取一般html的某些元素或属性，括起来用于传递参数。用flask的方法，如url_for()、request.form[]。  
#### blogCreate.html  
`<form method="post"></form>`：里面的内容可以传参了。  
`<input name="title" id="title" value="{{ request.form['title'] }}" required>`  
`<textarea name="body" id="body">{{ request.form['body'] }}</textarea>`  
`<input type="submit" value="Save">`  
#### blogUpdate.html  
已有数据时自动填充或没有数据：  
`<input name="title" id="title" value="{{ request.form['title'] or post['title'] }}" `  
`<textarea name="body" id="body">{{ request.form['body'] or post['body'] }}</textarea>`  
调用blog里的python函数，并传递参数：  
`<form action="{{ url_for('blog.delete', id=post['id']) }}" method="post">`  
### auth  
#### login.html  
`<input type="password" name="password" id="password" required>`：type="password" 会自动加密。加密函数在anth中定义。  
#### register.html  
除了标题和submit按钮的值外，其余的和login.html一模一样。
